# Cover Letter Rules

Never open with "I am writing to apply." Always reference company by name.
One specific achievement. Close with interview request. 1 page max.

Track-specific openers:
- A (Engineering): specific technical achievement from MT5 or Lobi Eco
- B (Product): business outcome, not technical detail
- C (Rail/Civils): name COSS/PC background early; reference NR standards
- D (Doc Control): EDMS platform name in first paragraph; Consepsys cert by name

EDMS detection is mandatory before any Track D cover letter is written.
