# Command: /score-job

Score a job description against all 4 CV tracks.

## Usage
/score-job [URL or paste JD text]

## What it does
Invokes the job-scoring skill. Produces a score for each track, selects the
best-fit track, states reasoning, and registers in the tracker.

## Output
- Track selected and score (e.g. "Track A — 4.2/5")
- Score for all 4 tracks
- Legitimacy assessment (Block G)
- Recommendation: apply / hold / skip
- Priority/urgent flag if applicable
