# Skill: Interview Prep

## Purpose
Generate a full interview prep pack when an interview is confirmed.

## When to invoke
INTERVIEW_INVITE classification from email monitor (Phase 2), or Charles says
"prepare for interview at [company]".

## How to apply
1. Pull the original JD used for the application
2. Pull the CV version that was submitted
3. Research the company: recent news, projects, technology stack, interviewers if known
4. Generate likely interview questions based on JD and submitted CV variant
5. Generate model answers using Charles's actual experience as evidence
   (never invent experience or metrics)
6. Generate company brief: key facts, recent projects, culture signals
7. Compile into prep pack
8. Deliver via Discord as formatted message + attached PDF (Phase 2)
9. Set 24-hour reminder before interview (Phase 2, deadline_manager)
